## Global

**Open File Search:** Cmd / Ctrl + shift + f

## Right Pane

**Next Tab:** Ctrl + ~  / Cmd + ~

**Prev Tab:** Ctrl + shift  + ~  /  Cmd + shift + ~

**To Rule Status Tab:** Ctrl + 1 /  Cmd + 1

**To Rule Editor Tab:**  Ctrl + 2 /  Cmd + 2

**Exit:** Esc


## Message Table

**Prev and Next:** ← /  →	


## Code Pane

**Undo Deleted Breakpoint:** Ctrl + Z / Cmd + Z
